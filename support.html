Support

Contact and support information for LegacyFixer self-service access.

Getting started

To start, use Sign in with GitHub. For support questions, email contact@legacyfixer.ai.

Support requests

For support, include the repository URL, status page URL if available, observed behavior, expected behavior, and whether pull request creation was enabled.

Security reports

For suspected security issues, email contact@legacyfixer.ai with the subject “LegacyFixer security report”.
