Support

Contact and support information for controlled access.

Access requests

To request access, use the branded contact address that will be published before public onboarding opens. Include one public Python repository URL.

Support requests

For support, include the repository URL, status page URL if available, observed behavior, expected behavior, and whether pull request creation was enabled.

Security reports

For suspected security issues, use the branded security contact address that will be published before public onboarding opens. Use the subject “LegacyFixer security report”.
