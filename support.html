Support

Contact and support information for LegacyFixer self-service access.

Getting started

To start, use Sign in with GitHub. For support questions, email contact@legacyfixer.ai.

Uninstall or remove access

You can uninstall LegacyFixer or remove repository access at any time from GitHub.

In GitHub, go to Settings → Applications → Installed GitHub Apps → LegacyFixer, then choose Uninstall or adjust repository access.

After the GitHub App is uninstalled or repository access is removed, LegacyFixer can no longer access those repositories through that installation. For workspace data removal requests, email contact@legacyfixer.ai.

Support requests

For support, include the repository URL, status page URL if available, observed behavior, expected behavior, and whether pull request creation was enabled.

Security reports

For suspected security issues, email contact@legacyfixer.ai with the subject “LegacyFixer security report”.
