Terms of self-service access

LegacyFixer is currently provided through self-service access for selected repositories.

Self-service access status

LegacyFixer provides GitHub sign-in, workspace creation, GitHub App installation, and trial-based access. During trial, repository connection may be limited.

Repository suitability

You must have the right to install the GitHub App on repositories you connect. Start with repositories you are comfortable evaluating in a dependency-maintenance workflow.

Review-first workflow

LegacyFixer does not automatically merge pull requests. Any proposed changes must be reviewed by the repository owner or maintainer before use.

No guarantee

LegacyFixer is provided for controlled evaluation. Scan results and proposed fixes may be incomplete, delayed, or unsuccessful. Repository maintainers remain responsible for reviewing dependency changes, security impact, and CI results.
